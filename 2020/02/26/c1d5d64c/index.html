发表于 2020-02-26 分类于 经验分享 ， Hexo博客

搭建博客

目前主流博客有两种，静态博客Hexo和动态博客WordPress。本文以Hexo为例来简单讲一下搭建过程。

前置准备

  • 首先服务器上要安装LNMP/LNMP环境
  • 如果是国内的服务器厂商（阿里云、腾讯云等）记得在安全组中开放端口
  • 服务器上安装好Git和Node.js

本机环境：CentOS 8

搭建过程

安装Git和Node.js

Git安装：

yum install git

Node.js安装:
根据自己的系统和自己所要求的版本去Github上查找
以v12为例

curl -sL https://rpm.nodesource.com/setup_12.x | bash -
dnf install nodejs

安装Hexo

在任意目录下执行
npm install -g hexo-cli
安装成功后执行
hexo init <folder> #<folder>为随意的文件名
cd <folder>
npm install

即可完成安装

注：npm源下载速度会比较慢，可以替换为淘宝源

npm config set registry http://registry.npm.taobao.org/

Hexo的默认端口为4000，记得在防火墙中开放4000端口
CentOS7+

firewall-cmd --add-port=4000/tcp --permanent
firewall-cmd --reload
