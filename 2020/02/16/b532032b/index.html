如何以非正常方法安装LAMP/LNMP

发表于 2020-02-16 更新于 2020-02-18 分类于 经验分享

在自己的服务器上安装LAMP/LNMP

拥有一台自己的服务器是一件很酷的事情，个人的能力决定了服务器的功能是否强大

使用一键包安装

为了方便小白用户的使用，已经有一些大佬已经整合出了一键安装包，我们可以访问它们的官网来执行安装

官网地址：LAMP LNMP

注意事项

  • 系统的最新版本，例如CentOS8等只能使用LNMP的稳定版
  • 网站根目录、PHP、MySQL等配置文件需要在官网文档中查询

    不按套路的安装方法

    此方法安装得不完全，追求稳定的同学还是使用一键包安装

安装Apache

yum install httpd #安装Apache
dnf install httpd #CentOS8通用
systemctl start httpd  #启动服务
systemctl enable httpd #设置为开机自启

安装MySQL/MariaDB

yum install mariadb mariadb-server #安装MariaDB(同MySQL)
systemctl start mariadb #启动数据库服务
systemctl enable mariadb #设为开机自启
mysql_secure_installation #进行数据库的设置
进入数据库的代码为
mysql -u root -p

安装PHP7

CentOS7下

方法一：
yum install epel-release #安装
yum install http://rpms.remirepo.net/enterprise/remi-release-7.rpm
yum install yum-utils
yum-config-manager --enable remi-php72
yum update #更新
yum install php72w #安装PHP
yum install php72-php-fpm php72-php-gd php72-php-json php72-php-mbstring php72-php-mysqlnd php72-php-xml php72-php-xmlrpc php72-php-opcache #安装拓展模酷爱

方法二：
rpm -Uvh https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm   
rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm    
yum install php72w

CentOS8下

dnf install epel-release
dnf install https://rpms.remirepo.net/enterprise/remi-release-8.rpm
remiPHP模块流的列表查看php
dnf module list php
启用php模块流
dnf module enable php:remi-7.2
安装PHP
dnf install php

测试PHP
在网站根目录下创建一个info.php的文件 

vi info.php
输入代码
<?php
phpinfo();
?>
使用:wq保存并退出

输入ip地址/info.php
