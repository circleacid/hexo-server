个人向网站整合

自己在闲逛的过程中发现的比较有趣、有用的网站或者一些软件，以此页面来记录下来，不定期更新，部分网站会补充相关教程。

学习类

  • JavaScript 算法与数据结构 Github
  • CS自学向书籍教程 计算机科学与技术
  • Goodies Github上部分内容的收集，包含网站开发、编程挑战等内容

    设计资源类

  • Font Awesome 网站图标
  • Pngimg 免费PNG图片资源，内容比较广泛
  • unDraw 免费插图资源下载，可根据颜色筛选
  • Prototypo 自行设计英文字体
  • Flexifont 同上，中文字体
  • Canva 在线设计网站，可设计logo、海报等

    软件类

  • Awesome Windows 字面意思，Windows上一些好用软件整理(Github)

影音类

  • PotPlayer 视频播放器，可导入视频源链接
  • SVP 插帧软件，可将本地24帧视频转换为60帧

    常用工具

  • Bandzip 压缩工具
  • Snipaste 截图工具
  • Everything 本地搜索软件
  • Listary 和Everything类似，比它更强大

    开发者向

  • VS Code
  • Typora Markdown文本编辑器

实用网站

  • 电子书全站下载
  • Bigjpg 图片放大

    教程类

学生党向

  • iData 中国知网文献专利等免费下载
  • 码云Gitee 中国版Github，可通过克隆的方式解决GitHub下载慢的问题
  • 力扣LeetCode 编! 程! 练! 习!
